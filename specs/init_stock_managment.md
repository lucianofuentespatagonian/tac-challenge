# Stock Managment

## Project Overview
Build a stock managment interface MVP.


**Tech Stack:**
- Frontend: Next.JS (modern dev experience, type safety, zero framework complexity)
- Backend: Next.JS (fast dependency management, type safety, auto docs)
- Database: localstorage (portable, zero setup)

**MVP Core Flow:**
1. User managment
2. Product managment
3. Stock managment
4. Stores managment
